## 一、简介
这是一个用来下载21世纪报的爬虫
## 二、使用方式
### 1.命令行模式
将cookie放到cookie.txt内，运行i21-paper-download-pdf即可
> cookie 获取方式，请参考[百度](https://www.baidu.com/s?wd=%E8%8E%B7%E5%8F%96cookie/)
### 2.GUI模式
直接运行main.py，根据提示填写cookie、选择下载路径即可
